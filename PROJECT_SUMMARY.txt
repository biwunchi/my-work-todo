================================================================================
KEY COMPONENTS EXPLAINED
================================================================================

1. CalendarView.tsx
   - Displays a full month calendar with day grid
   - Shows task counts for each day
   - Hover preview of task titles with priorities
   - Navigate between months with prev/next buttons
   - Color-coded priority badges

2. WeeklyView.tsx
   - Shows 7 columns (Mon-Sun)
   - Large task count display per day
   - Preview of first 3 tasks per day
   - Easy week navigation
   - Responsive grid layout

3. DailyView.tsx
   - Shows all tasks for selected date
   - Full task details with descriptions
   - Edit and delete buttons for each task
   - Priority badge display
   - Created timestamp display

4. TaskForm.tsx
   - Reusable form for adding and editing tasks
   - Fields: Title, Description, Date, Priority
   - Form validation (title required)
   - Clear form after successful add
   - Works in edit mode with pre-filled data

5. useTasks Hook
   - Central state management for tasks
   - CRUD operations (Create, Read, Update, Delete)
   - Supabase integration
   - Loading and error states
   - Automatic fetching on mount

6. ThemeToggle.tsx
   - Toggle between light and dark modes
   - Saves preference to localStorage
   - Respects system preference on first load
   - Uses Tailwind's dark class selector
