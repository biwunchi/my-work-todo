# 001 - Calm Editorial Design

## 설계 철학

**차분하고 신뢰감 있는 환경** 조성을 목표로 합니다. 인쇄된 잡지나 신문의 편집 미학에서 영감을 받아, 고급스럽고 문서 지향적인 느낌을 제공합니다.

### 핵심 원칙
- **여유있는 공간**: 넉넉한 여백과 낮은 밀집도로 편안함 제공
- **타이포그래피 중심**: Georgia 세리프 폰트로 전문성과 신뢰감 표현
- **부드러운 상호작용**: 미묘한 호버 효과와 부드러운 전환
- **중립적 색상**: 베이지, 회색, 검은색으로 디스트랙션 최소화

## 주요 선택 사항

1. **레이아웃**: 2컬럼 (달력 메인 + 우측 사이드바)
2. **색상 팔레트**:
   - 배경: #faf9f7 (따뜻한 베이지)
   - 텍스트: #2c2c2c (다크 회색)
   - 액센트: #a89f99 (은은한 갈색)
3. **폰트**: Georgia, Garamond, serif
4. **간격**: 넉넉한 패딩과 마진 (2rem~3rem)
5. **태스크 표시**: 작은 원형 도트 (달력의 각 날짜에)

## 강점

✅ **높은 가독성**: 넓은 공간과 명확한 타이포그래피  
✅ **우아한 느낌**: 전문적이고 신뢰감 있음  
✅ **집중력 향상**: 불필요한 시각 자극 최소화  
✅ **장시간 사용성**: 눈이 덜 피로하는 디자인  
✅ **계획/분석 작업**: 깊이 있는 사고 필요시 적합

## 약점

❌ **정보 밀집도 낮음**: 한 화면에 표시되는 정보가 적음  
❌ **모바일 최적화 부족**: 큰 여백이 작은 화면에서 낭비됨  
❌ **빠른 작업성**: 컴팩트한 UI를 원하는 사용자에게는 불편  
❌ **현대성 부족**: 미니멀/모던 디자인 선호자에게는 구식으로 보일 수 있음

## 최적 사용 시나리오

- 📋 **주간/월간 계획 수립**
- 📖 **장기 프로젝트 추적** (세부 사항과 설명이 필요한 경우)
- 💼 **전문가용 도구** (비즈니스, 학술 환경)
- 🧘 **스트레스 없는 일정 관리** (다급하지 않은 환경)
- 📝 **많은 설명 텍스트 필요** (간단한 제목보다 세부 정보 중요)

## 기술 스택

- **스타일링**: 인라인 CSS (외부 의존성 없음)
- **폰트**: 시스템 기본 세리프 폰트
- **상호작용**: Vanilla JavaScript
- **호환성**: 모든 최신 브라우저

## 확장 가능성

이 디자인을 기반으로 다음을 추가할 수 있습니다:
- 다크 모드 (별도의 색상 팔레트)
- 드래그 앤 드롭 태스크 재정렬
- 리치 텍스트 에디터 (설명 필드)
- 주간/월간 요약 차트
- 태스크 분류 및 필터링
